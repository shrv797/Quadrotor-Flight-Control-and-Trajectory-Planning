# Quadrotor Flight Control

🚧 **Project Status: Active Development (Work in Progress)**

This repository contains an autonomous quadrotor flight control project developed using Python and ROS2.

The project focuses on implementing core flight control algorithms and trajectory generation for autonomous UAV operations. New features and improvements are continuously added as development progresses.

## Features

- Autonomous Takeoff
- Autonomous Landing
- Geometric Trajectory Generation
  - Line
  - Square
  - Triangle
- Waypoint Navigation
- ROS2-based Architecture
- Python Implementation

## Technologies

- Python
- ROS2
- MAVROS

## Project Structure
